# do-better

### Gamified Habit tracker

- Leveling system to help promote growth and continued use.
- Featuring social rankings among friends and all users.
- Achievements for long streaks of completing goals and increasing in rank.
- Reminders to keep from losing streaks.
- Premium features include curated diet plans and workout plans. (A.I?)
